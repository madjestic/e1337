# A 1984 Elite-inspired space-sim game

## Building and running:
```bash
$ cabal new-build
$ gpu ./run.sh
```

## Rebuilding assets:
```basj
$ cd ./resources
$ make
```

## Working with REPL:
```bash
$ cabal new-repl exe:e1337
```
